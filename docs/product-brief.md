# fortune-lab Product Brief
> 작성일: 2026-04-06 | /council 결과

## 서비스 한 줄 요약
사주와 이름 한자 풀이를 한 플로우에서 제공하는, 2030을 위한 운세 서비스

## 타깃
- 20-30대, 자신의 이름/사주에 관심 있는 일반인
- 특히 결과를 SNS에 공유하는 성향의 사용자

## 핵심 차별점 (UVP)
기존 서비스는 사주와 이름풀이가 분리되어 있음.
fortune-lab은 생년월일 입력 → 이름 한자 선택 → 통합 해석을 한 흐름으로 제공.

## 시장 맥락
- 국내 점술 시장 약 1.4조원
- 점신(테크랩스) 연매출 830억 (YoY +58%)
- 20-30대가 운세 소비의 80% 차지
- 이름풀이 경쟁사 대부분 구형 UI, 사주와 분리, 무료 위주

## 당장 개선 우선순위

### 1순위 — 한자 선택 UX
- 문제: 한자를 모르는 사용자가 선택 단계에서 이탈 위험
- 해결: 자동 추천 한자 기본 선택 + 원하면 직접 변경 옵션

### 2순위 — 결과 공유 기능
- 문제: 바이럴 루프 없으면 트래픽 정체
- 해결: 결과 카드 이미지 다운로드 또는 OG 공유 링크
- 목표: "내 결과 공유" CTA + "친구 결과 보기" 재진입

### 3순위 — 수익화 훅 1개
- 결과 하단: "상세 대운 풀이 전체 보기 (₩3,900~₩5,900)"
- 무료: 요약 결과 + 공유
- 유료: 대운 전체 + 이름별 상세 분석

## 수익 모델
- 프리미엄 (결과 상세 유료 잠금) 우선 적용
- 손익분기: 월 유료 전환 50명 수준 (서버리스 구조상 고정비 낮음)

## 검증이 필요한 핵심 가정
1. 한자 선택 단계 이탈률이 낮은가? → 퍼널 분석
2. 결과 공유가 실제로 발생하는가? → 공유 버튼 클릭률 추적

## MVP 스코프 (지금 당장)
- [ ] 한자 자동 추천 기본값 적용
- [ ] 결과 공유 카드 or OG 이미지 생성
- [ ] 유료 전환 포인트 1개 추가
